Repositorio de prácticas de aula en **JavaScript ES6+**: ejercicios de programación web que cubren la manipulación del DOM, programación orientada a eventos, consumo de APIs REST con fetch, programación asíncrona con Promises/async-await y patrones de diseño aplicados a frontend.

## Conceptos cubiertos

| Tema | Tecnología |
|---|---|
| Manipulación del DOM | querySelector, innerHTML, classList |
| Eventos | addEventListener, delegación de eventos |
| Asincronía | Promise, async/await, fetch API |
| ES6+ Features | Arrow functions, destructuring, spread/rest, modules |
| Almacenamiento | localStorage, sessionStorage |
